# دليل خدمات برنامج متين العلمي — حسب كل مستخدم

---

## 👑 الإدارة (Admin)

### الوصول
- لوحة الإداريات الكاملة (`admin.html`)
- جميع صفحات الموقع بدون قيود

### الطالبات
- عرض جميع الطالبات المسجلات وأصدقاء متين
- إضافة طالبة جديدة (يدوياً أو جماعياً)
- تعديل بيانات أي طالبة
- تحديد حالة المقابلة (تمت / لم تتم)
- تغيير حالة الطالبة (مستجدة / بنات متين)
- حذف حساب أي مستخدم نهائياً من Firestore و Firebase Auth
- تصدير بيانات الطالبات لملف Word مع اختيار الأعمدة والتقسيم بالوقت

### المواد العلمية
- تعديل المادة الرئيسية نفسها (العنوان الفرعي، الوصف، المحاور)
- إضافة محتوى (PDF، فيديو، مقال، رابط) لأي مادة
- تعديل وحذف أي محتوى مضاف

### المكتبة
- إضافة وتعديل وحذف في كل الأقسام:
  - مكتبة متين
  - المسار الإثرائي
  - بودكاست تبصرة
  - دورات متنوعة

### الرسائل
- مراسلة أي مستخدم
- إرسال صور وتسجيلات صوتية

### الأخبار
- إضافة وتعديل وحذف الأخبار والإعلانات

### الجدول الدراسي
- تحديد مواعيد المقابلات وتعيين الجداول

---

## 🛡️ المشرفة (Supervisor)

### الوصول
- لوحة المشرفة (`supervisor.html`)
- نفس صلاحيات الإدارة تقريباً ماعدا حذف الحسابات

### المواد العلمية
- إضافة وتعديل وحذف المحتوى في كل المواد
- **لا تستطيع** تعديل المادة الرئيسية نفسها

### المكتبة
- إضافة وتعديل وحذف في كل الأقسام

### الرسائل
- مراسلة أي مستخدم
- إرسال صور وتسجيلات صوتية

### الطالبات
- عرض بيانات الطالبات
- متابعة حالات المقابلات

---

## 👩‍🏫 المعلمة (Teacher)

### الوصول
- صفحة مادتها الخاصة فقط (تفسير / فقه / عقيدة / حديث / مقرأة متين)

### المواد العلمية
- إضافة محتوى لمادتها فقط
- تعديل وحذف المحتوى الخاص بمادتها فقط
- لا تستطيع التعديل على مواد الأخريات

### طالباتها
- عرض قائمة الطالبات المسجلات في مادتها
- متابعة درجاتهن وحضورهن

### الرسائل
- مراسلة الطالبات والإدارة والمشرفة
- إرسال صور وتسجيلات صوتية

### المكتبة
- مشاهدة فقط

---

## 📖 الطالبة (Student / بنات متين)

### الوصول
- ملفها الشخصي (`student.html`)
- المواد التي التحقت بها فقط

### المواد العلمية
- مشاهدة المحتوى المضاف للمواد التي سجّلت فيها فقط
- التسجيل في المواد المتاحة

### ملفها الشخصي
- عرض درجاتها
- عرض شهاداتها وإجازاتها
- الجدول الدراسي الخاص بها

### الرسائل
- مراسلة المعلمات والمشرفة والإدارة فقط
- **نصوص فقط** — لا يمكن إرسال صور أو تسجيلات

### المكتبة
- مشاهدة كل الأقسام

### الأخبار
- مشاهدة الأخبار والإعلانات

---

## 🌺 أصدقاء متين (Mateen)

### الوصول
- محدود — زوار مهتمون بالبرنامج

### المواد العلمية
- مشاهدة وصف المواد فقط
- **لا يمكن** الالتحاق بالمواد

### الرسائل
- مراسلة الإدارة والمشرفة فقط
- نصوص فقط

### المكتبة
- مشاهدة كل الأقسام

### الأخبار
- مشاهدة الأخبار والإعلانات

---

## 👁️ الزائر (غير مسجّل)

- مشاهدة الصفحة الرئيسية
- مشاهدة صفحة "عن البرنامج"
- التسجيل الجديد في البرنامج
- **لا يمكن** الوصول لأي محتوى داخلي
